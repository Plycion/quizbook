    Mission Statement

    • Quizbook is a minimalistic study tool governed by spaced repetition algorithms packaged in a gamified environment. It provides visual feedback that is both immediate and intuitive, using a pool of test questions collaboratively constructed. Tests and practice sessions are generated uniquely according to the user's progress and history.
